Eclipse Vorto Web Editor

Web based code editor for Eclipse Vorto IoT languages

Github
May 2016 - Aug 2016

INTRODUCTION
Eclipse Vorto is an open source tool that allows to create and manage technology agnostic, abstract device descriptions, so called information models. These information models could be created using the Eclipse Vorto plugin. However not everyone was comfortable using this plugin and there was a need to create a web editor.I developed this web editor as my project during Google Summer of Code, 2016.
